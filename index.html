⚠ Pneumatic Stage Removed — PH10 / DF0 Decommissioned — Air tubing capped and isolated

The 4–20 mA signal from the temperature converter now routes directly to electronic instrumentation. No pneumatic conversion stage is required. The Foxboro pneumatic controllers and recorders are replaced by electronic equivalents. This is Step 3 in the phased migration strategy.

✅ Benefits Gained

  • Eliminates all pneumatic equipment and maintenance
  • No instrument air consumption for conversion
  • Improved accuracy — electronic vs. pneumatic
  • Electronic data storage, trending, alarm logging
  • Ethernet communication to historian/SCADA
  • PID control with auto-tune capability

📋 New Equipment

  • Yokogawa GX10 / GX20 Paperless Recorder
  • Yokogawa UT55A or Siemens PID Controller
  • 4–20 mA wiring to new panel instruments
  • Ethernet switches for data network
  • Historian software (e.g. Yokogawa PRM)
Upgrade Plan 2B — Direct Input to GX10 (No Converter Required) OPTIMAL UPGRADE

The Yokogawa GX10 Paperless Recorder and modern electronic controllers accept RTD (Pt100) and Thermocouple (TC) signals directly at their input terminals. This eliminates the intermediate conversion stage entirely — no VJ series converter, no DF0 E/P card, no PH10 is needed.

🏆 Why Direct Input is Better

  • Eliminates entire converter layer (VJ / DCP2)
  • No intermediate 4–20 mA loop errors
  • Fewer components = higher system reliability
  • Lower installation cost
  • Fastest path from sensor to digital record

📋 GX10 Input Capabilities

  • Thermocouples: K, J, T, E, R, S, B, N, W
  • RTD: Pt100 (3-wire), Pt200, Pt500, JPt100
  • DC Voltage: 20 mV to 100 V
  • DC Current: 4–20 mA (with shunt)
  • Accuracy: ±0.05% of span typical
  • Up to 100 channels expandable

⚠ When You Still Need Converters

  • Long cable runs >500m (4–20 mA better for noise)
  • Galvanic isolation requirements
  • Existing DCS with only 4–20 mA inputs
  • HART communication requirements

📋 Alarm Response Procedures — DCP2 System

  • RTD/TC Open Circuit: Check 3-wire field connections at terminal strip → Inspect DR5/DT5 card LED (solid red = card fault) → Hot-swap replace card, no shutdown required
  • Air Supply Loss: Check 20 PSI header pressure at Rc 3/8" inlet → Inspect PPU-16 isolation valves → Check upstream filter/regulator → Verify air compressor status
  • Card Failure: Remove and re-seat card in DCP2 slot → Check backplane connector pins → Replace with spare DR5/DT5/DF0 card (hot-swap on DCP2) → Verify output using monitor terminals
  • Temperature HH: Verify field sensor calibration → Cross-check with independent measurement → Check span setting via DSC2/VJ77 → Initiate process response per operating procedure
  • Power Failure: Check X039 PSU output LED → Verify 110 VAC input breaker → Inspect 24 VDC bus fuse → Check backplane terminal voltage
Phased Migration Roadmap — RASCO Ethylene Plant
Phase 1 — Now

⚠ Legacy Operation

34C-AP converters in service. No spare parts. Mechanical drift. Immediate risk to production continuity.

Phase 2 — Short Term

🔌 Modular Replacement

DCP2 nest (Alt 1) preferred. Loop-by-loop migration. Existing Foxboro pneumatic equipment preserved.

Phase 3 — Medium Term

📊 Electronic Upgrade

Remove PH10/DF0 pneumatics. Route 4–20 mA to GX10 recorders and electronic PID controllers.

Phase 3B — Optional

🎯 Direct RTD/TC Input

Upgrade to GX10 with direct TC/RTD inputs. Eliminate converter layer entirely. Maximum accuracy.

Phase 4 — Long Term

🌐 DCS / SCADA

CENTUM VP full implementation. Centralized HMI, historian, alarm management. Industry 4.0 ready.

Step 1 — Initial Phase (Zero Production Impact)

  • Install empty 16-slot DCP2 chassis into existing 19" cabinet during routine operation
  • Wire 110 VAC supply and 20 PSI air headers to nest central inputs (Rc 3/8")
  • Wire DSC2 RS-485 Modbus to communication terminal for future DCS connection
  • Cover all empty slots with blank faceplates

Step 2 — Execution Phase (Loop-by-Loop, Scheduled Maintenance Windows)

  • Select individual loops (e.g., TR-1001) during scheduled maintenance isolation
  • Remove legacy Foxboro 34C-AP surface enclosure from wall mount
  • Land existing 3-wire RTD or TC field cables onto designated DCP2 slot terminals
  • Slide in DR5 (RTD) or DT5 (TC) input card + DF0 (E/P) output card as loop pair
  • Route new 3–15 PSI tube from DF0 Rc 1/4" output to existing Foxboro controller/recorder
  • Verify calibration via monitor terminals — adjust zero/span on card if needed

Step 3 — Density Optimization

  • 1 × DCP2 16-slot nest can replace up to 8 individual Foxboro wall-mounted converter boxes (loop pairs)
  • Mixed RTD and TC loops share the same nest with correct card types per slot
  • VJU7 universal card (Alt 2) recommended for spare parts standardization — one SKU for both RTD and TC
  • DSC2 card enables reading all 16 channels via single RS-485 serial port
