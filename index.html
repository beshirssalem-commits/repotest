📋 Alarm Response Procedures

  • RTD Open Circuit: Check 3-wire connections at terminal → inspect DR5/VJR6 card LED → replace card if LED solid red (hot-swap, no shutdown)
  • Air Supply Loss: Check 20 PSI header at manifold → inspect PPU-16 isolation valves → check filter/regulator upstream
  • Card Failure: Remove and re-seat card → check backplane connector → replace with spare (hot-swappable on DCP2/VJ)
  • Temperature HH: Verify sensor calibration → check loop span setting → initiate process response per operating procedure
  • Power Failure: Check X039 PSU output LED → check 110 VAC input breaker → check 24 VDC bus fusing
Phased Migration Roadmap
Phase 1 — Now

Legacy Operation

34C-AP R/P converters in service. No spare parts. Mechanical drift risk. Immediate action required.

Phase 2 — Short Term

Modular Replacement

Install DCP2 nest (Alt 1) or VJ+PH10 (Alt 2). Loop-by-loop migration. Preserve Foxboro controllers & recorders.

Phase 3 — Medium Term

Electronic Upgrade

Remove PH10/DF0 pneumatic stage. Route 4–20 mA to GX10 recorders and electronic controllers.

Phase 4 — Long Term

DCS / SCADA

CENTUM VP / PCS7 implementation. Modbus gateway. Centralized HMI, historian, alarm management.

Step 1 — Initial Phase (Zero Production Impact)

  • Install empty 16-slot DCP2 or VJ+X018 chassis during routine operation
  • Wire 110 VAC supply and 20 PSI air headers to nest central inputs
  • Cover empty slots with blank faceplates — system fully installed but idle

Step 2 — Execution Phase (Loop-by-Loop, Scheduled Windows)

  • Select individual loops (e.g., TR-1001) during scheduled maintenance windows
  • Remove legacy Foxboro surface enclosure from wall/rack
  • Land existing 3-wire RTD field cables onto designated nest slot terminals
  • Slide in DR5 (RTD) or DT5 (TC) input card + DF0 output card into slot pair
  • Route new 3–15 PSI tube from DF0 card output to Foxboro controller/recorder

Step 3 — Density Optimization

  • 1 × DCP2 16-slot nest replaces up to 9 individual Foxboro wall-mounted boxes
  • Mixed RTD and TC loops can share the same nest chassis
  • VJU7 universal card recommended — one spare replaces both RTD and TC channels
