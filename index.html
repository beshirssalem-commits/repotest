📋 Alarm Response Procedures

  • RTD Open Circuit: Check 3-wire connections at terminal → check DR5/VJR6 card LED → replace card if LED solid red
  • Air Supply Loss: Check 20 PSI header pressure at manifold → check PPU-16 isolation valves → check instrument air filter/regulator
  • Card Failure: Remove and re-seat card → check backplane slot → replace with spare (hot-swap capable)
  • Temperature HH: Check sensor calibration → verify loop span setting → initiate process response per operating procedure
  • Power Failure: Check X039 PSU output → check 110 VAC input → check 24 VDC bus fusing
Phased Migration Roadmap
Phase 1 — Now

Foxboro Legacy Operation

34C-AP R/P converters in service. Risk: no spare parts, mechanical drift. Immediate action required.

Phase 2 — Short Term

Modular Replacement

Install DCP2 nest (Alt 1) or VJ+PH10 (Alt 2). Loop-by-loop migration. Preserve existing controllers & recorders.

Phase 3 — Medium Term

Electronic Upgrade

Remove PH10/DF0 stage. Route 4–20 mA to GX10 recorders and electronic controllers. Retire pneumatics.

Phase 4 — Long Term

DCS / SCADA Migration

CENTUM VP / PCS7 implementation. Modbus gateway. Centralized HMI, historian, alarm management.

Phased Implementation Strategy

Step 1 — Initial Phase (Zero Production Impact)

  • Install empty 16-slot DCP2 or VJ+X018 chassis during routine operation
  • Wire 110 VAC supply and 20 PSI air headers to nest central inputs
  • Cover empty slots with blank faceplates

Step 2 — Execution Phase (Loop-by-Loop, Scheduled Windows)

  • Select individual loops (e.g., TR-1001) during scheduled maintenance windows
  • Remove legacy Foxboro surface enclosure
  • Land existing 3-wire RTD field cables onto nest slot terminals
  • Slide in DR5 (RTD) or DT5 (TC) input card + DF0 output card
  • Route new 3–15 PSI tube to Foxboro controller/recorder

Step 3 — Balance Racks (Density Optimization)

  • 1 × DCP2 16-slot nest can replace up to 9 individual Foxboro wall boxes
  • Mixed RTD and TC loops can share the same nest
  • VJU7 universal card recommended for single spare-part strategy
